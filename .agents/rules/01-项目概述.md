# 项目概述

## 项目定位

这是一个相亲匹配管理系统，旨在为用户提供便捷的相亲对象匹配服务。系统支持用户管理、信息展示、智能匹配等功能。

## 技术栈

| 领域 | 技术 | 说明 |
| ---- | ---- | ---- |
| 框架 | Next.js 16 | React 全栈框架 |
| 语言 | TypeScript | 类型安全 |
| UI 组件库 | Radix UI | 无样式访问性组件库 |
| 样式 | SCSS + SCSS Modules | 模块化样式管理 |
| 表单验证 | Zod | Schema 验证 |
| HTTP 客户端 | Axios | 请求封装 |
| 日期处理 | Day.js | 轻量级日期库 |
