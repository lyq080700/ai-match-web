# 文档规范

## 注释规范

- 使用 JSDoc 风格的注释（`/** */`），包含必要的描述信息
- 注释应该解释"为什么"而不是"是什么"，避免冗余注释

## 注释类型

| 类型 | 内容 |
|------|------|
| 文件头部 | 说明文件用途和主要功能 |
| 函数/方法 | 说明功能、参数、返回值 |
| 复杂逻辑 | 解释关键算法、业务逻辑或特殊处理 |
| 类型定义 | 说明接口、类型、枚举的含义和用途 |
