## 使用说明

1. 根据需要的规范类型，点击对应的模块链接查看详细内容
2. 所有规范文件都包含 `alwaysApply: true` 标记，确保规范自动应用
3. 详细示例与落地步骤见 `.cursor/skills/` 目录下的技能文件
