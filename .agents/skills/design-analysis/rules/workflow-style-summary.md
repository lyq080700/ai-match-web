# 第三步：样式规范汇总

## 概述

在分析清单中单独成章，便于开发与验收统一使用。汇总设计稿中的颜色、字体、圆角、间距、阴影等样式规范，形成可复用的样式指南。

## 汇总内容

### 颜色规范

提取设计稿中的主要颜色，按用途分类：

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主文本 | `#13131c` | 主要文字颜色 |
| 次要文本 | `#13131c73` | 次要文字颜色（含透明度） |
| 边框 | `#13131c40` | 边框颜色 |
| 背景 | `#ffffff` | 页面背景色 |
| 按钮主色 | `#1890ff` | 主要按钮颜色 |
| 按钮禁用 | `#d9d9d9` | 禁用状态颜色 |

### 字体规范

按用途汇总字体族、字号、字重、行高：

| 用途 | 字体族 | 字号 | 字重 | 行高 | 示例 |
|------|--------|------|------|------|------|
| 大标题 | PingFang SC | 24px | 600 | 32px | 页面主标题 |
| 副标题 | PingFang SC | 18px | 500 | 26px | 区块标题 |
| 正文 | PingFang SC | 14px | 400 | 20px | 普通文本 |
| 说明文字 | PingFang SC | 12px | 400 | 18px | 辅助说明 |
| 按钮文字 | PingFang SC | 14px | 500 | 20px | 按钮内文字 |

### 圆角规范

按元素类型汇总圆角值：

| 元素类型 | 圆角值 | 说明 |
|----------|--------|------|
| 卡片 | 12px | 卡片容器 |
| 按钮 | 8px | 按钮圆角 |
| 输入框 | 8px | 输入框圆角 |
| 图片 | 8px | 图片圆角 |
| 标签 | 4px | 标签圆角 |

### 间距规范

按层级汇总间距值：

| 层级 | 间距值 | 说明 |
|------|--------|------|
| 页面边距 | 132px | 页面左右边距 |
| 区块间距 | 24px | 区块之间的垂直间距 |
| 组件内 gap | 12px | 组件内部元素间距 |
| 组件内 padding | 16px | 组件内边距 |

### 阴影与效果

汇总阴影和特殊效果：

| 效果类型 | 参数 | 说明 |
|----------|------|------|
| 卡片阴影 | `0 2px 8px rgba(0,0,0,0.1)` | 卡片阴影 |
| 按钮阴影 | `0 1px 4px rgba(0,0,0,0.1)` | 按钮阴影 |
| 背景模糊 | `backdrop-filter: blur(17.5px)` | 背景模糊效果 |

## 汇总方法

1. **从区域分析中提取**：在第二步「区域与元素提取」时，记录每个区域的样式信息
2. **按类型分类**：将相同类型的样式值归类汇总
3. **去重合并**：相同的样式值只记录一次，避免重复
4. **标注用途**：明确每个样式值的用途和使用场景

## 使用建议

- **开发时**：参考样式规范汇总，确保实现时使用统一的样式值
- **验收时**：对照样式规范汇总，检查实现是否符合设计稿规范
- **维护时**：设计稿变更时，同步更新样式规范汇总
