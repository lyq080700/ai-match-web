## 层级比对规则（P0）

**P0 级要求**：层级丢层、顺序颠倒为 P0，验收时不得遗漏。

**比对内容**：
- **层级**：从外到里的嵌套、父子/兄弟、先后与叠放顺序；是否丢层、顺序颠倒。

**验收顺序**（必须按以下顺序进行，避免漏检）：
1. **从上到下**：从页面最上方区域逐块检查到最下方，顺序与布局一致。
2. **从左到右**：同一行或同一层级内，按从左到右依次检查，不跳过。
3. **从外到里**：每个区块内按「外层容器 → 内层容器 → 叶子元素」逐层检查。

**层级检查要点**：
- 检查嵌套关系是否正确（谁包谁、父子/兄弟关系）
- 检查区块的先后顺序是否正确
- 检查叠放顺序（z-index）是否正确
- 检查是否有丢层（缺少必要的容器）
- 检查顺序是否颠倒（元素顺序与设计稿不一致）

**验收时检查**：
- [ ] 从外到里的嵌套关系是否正确？
- [ ] 父子/兄弟关系是否正确？
- [ ] 区块的先后顺序是否正确？
- [ ] 叠放顺序是否正确？
- [ ] 是否有丢层？
- [ ] 顺序是否颠倒？
