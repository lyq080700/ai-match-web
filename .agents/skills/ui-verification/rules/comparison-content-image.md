## 图片内容比对规则（P0）

**P0 级要求**：图片缺失、布局错为 P0，验收时不得遗漏。

**比对内容**：
- **内容 - 图片**：所有图片/占位图区域是否存在、尺寸与布局（含多图拼接）是否正确。

**验收顺序**（必须按以下顺序进行，避免漏检）：
1. **从上到下**：从页面最上方区域逐块检查到最下方，顺序与布局一致。
2. **从左到右**：同一行或同一层级内，按从左到右依次检查，不跳过。
3. **从外到里**：每个区块内按「外层容器 → 内层容器 → 叶子元素」逐层检查。

**图片检查要点**：
- 检查所有图片/占位图区域是否存在
- 检查图片尺寸是否正确
- 检查图片位置是否正确
- 检查多图拼接方式是否正确（1/2/3/4 图拼接）
- 检查图片比例是否正确
- 检查占位图尺寸与布局是否正确

**验收时检查**：
- [ ] 所有图片/占位图区域是否存在？
- [ ] 图片尺寸是否正确？
- [ ] 图片位置是否正确？
- [ ] 多图拼接方式是否正确（1/2/3/4 图拼接）？
- [ ] 图片比例是否正确？
- [ ] 占位图尺寸与布局是否正确？
