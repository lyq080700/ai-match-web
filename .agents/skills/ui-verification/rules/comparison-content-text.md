## 文字内容比对规则（P0）

**P0 级要求**：文字缺失、错字为 P0，验收时不得遗漏。

**比对内容**：
- **内容 - 文字**：所有可见文案是否存在、内容是否正确（欢迎语、标题、副文案、按钮/占位/Tab 等）。

**验收顺序**（必须按以下顺序进行，避免漏检）：
1. **从上到下**：从页面最上方区域逐块检查到最下方，顺序与布局一致。
2. **从左到右**：同一行或同一层级内，按从左到右依次检查，不跳过。
3. **从外到里**：每个区块内按「外层容器 → 内层容器 → 叶子元素」逐层检查。

**文字检查要点**：
- 检查所有可见文案是否存在（欢迎语、标题、副标题、说明、按钮/链接文案、占位文案、Tab 文字等）
- 检查文案内容是否正确（是否有错字、漏字）
- 检查文案是否完整（是否被截断）
- **特别注意**：顶部或首屏的大字号文案（如欢迎语）最容易在「只记了下面小字」时被漏掉，必须优先按从上到下扫到并记录

**验收时检查**：
- [ ] 所有可见文案是否存在？
- [ ] 文案内容是否正确（是否有错字、漏字）？
- [ ] 文案是否完整（是否被截断）？
- [ ] 顶部或首屏的大字号文案是否已检查？
